OSL Common Utils
================

A suite of commonly used java utilities, frequently used within the 'open systems landscape'

## Included Utilities

 * Email
 * Exception
 * Hystrix
 * Mongo
 * Queue
 * Validation

## Requirements

 * Java JDK 1.8+
 * Maven 3

## How to build it

To build this project execute the following command:

```bash
  mvn clean install
```
